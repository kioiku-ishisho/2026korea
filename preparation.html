📋 行前準備清單

出發前必備物品與注意事項

📄 證件文件

  • ✓ 護照（有效期6個月以上）
  • ✓ 機票確認單
  • ✓ 飯店預訂確認單
  • ✓ 旅遊保險單
  • ✓ 護照影本（備用）

💰 金錢相關

  • ✓ 現金（台幣或美金，用於換匯）
  • ✓ 信用卡（Visa/Mastercard）
  • ✓ 提款卡（確認海外提款功能）
  • ✓ 零錢包（小額韓元）

📱 電子設備

  • ✓ 手機及充電器
  • ✓ 行動電源
  • ✓ 轉接頭（韓國使用Type C/F，220V）
  • ✓ 相機（可選）
  • ✓ 耳機

👕 衣物用品

  • ✓ 保暖衣物（1月首爾較冷，約-5°C~5°C）
  • ✓ 羽絨外套、圍巾、手套
  • ✓ 舒適的步行鞋
  • ✓ 換洗衣物
  • ✓ 個人衛生用品

💊 健康用品

  • ✓ 個人常用藥品
  • ✓ 感冒藥、腸胃藥
  • ✓ 口罩
  • ✓ 濕紙巾、乾洗手

📲 手機應用程式

  • ✓ Naver Map / Kakao Map（地圖）
  • ✓ Papago（翻譯軟體）
  • ✓ Kakao T（計程車）
  • ✓ Seoul Metro（地鐵）
  • ✓ 離線地圖（備用）

💡 重要提醒

出發前檢查事項

  • 確認護照有效期至少6個月以上
  • 確認機票和飯店預訂資訊
  • 購買旅遊保險
  • 確認手機國際漫遊或購買當地 SIM 卡
  • 下載必要的手機應用程式
  • 準備轉接頭（韓國使用Type C/F插頭）
  • 確認信用卡海外使用功能已開啟
  • 準備常用藥品和個人衛生用品
